*The output should be two <u>capital</u> letters with a dot separating them.*

```Java
function abbrevName(name){
  let arr_name = name.split(" ");
  let abbrev = arr_name.map((word) => word[0].toUpperCase()).join('.');
  return(abbrev)
}
```
